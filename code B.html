Code B

Our Program

Code B is one of many opertunties that allows future programers or games design to build a path for themselves or to even sharpen their skills. BGHS STEM Alumni founded the program and are working diligently to create informative tutorials on how to design games. You will learn step by step on how to design and code games with informative tutorials lasting 5-15 minutes. Learn what it takes to make a professional or fun game with NASA software engineer Julian Vu and experience the fun process of game design
