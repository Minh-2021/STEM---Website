Code B

Our Program

Code B is one of many opportunities in Bolsa Grande Highschool's STEM club that allows future programmers or games design to build a path for themselves or sharpen their skills in programming. BGHS STEM Alumni founded the program and are working diligently to create informative tutorials on how to code, in order to design games. You will learn step by step on how to design and code games with informative tutorials lasting 5-15 minutes. Learn what it takes to make a professional or fun game with NASA software engineer Julian Vu and experience the fun process of game design

Developers of Code B

Julian Vu

Julian Vu got the idea for Code B when he noticed that Bolsa did not have any computer science class. He hopes to get more students to learn how to code in order to prepare them for an increasingly technological world. He is now the mentor for the program and is in all of the video tutorials for Code B

Danny Vo

Danny Vo was a Junior at the time when he realized the lack of coding classes or clubs on Bolsa's campus, as well as the fact that many Bolsa Grande Graduates where graduating as computer science majors, without ever having the chance to code before. He wanted to help students learn how to code in a comfortable environment so they could achieve greater heights in college.
